# 🏠 HAKiosk Integration for Home Assistant

Esta integração permite controlar e monitorizar o teu tablet ou telemóvel Android que corre a app **HAKiosk v3.4+**.

## ✨ Funcionalidades
- 🔋 Sensores de Bateria, Memória e Estado de Carga.
- 📺 Entidade de Câmara para ver o ecrã do tablet em tempo real.
- 🔊 Serviço de Voz (TTS) direto para o dispositivo.
- 🔗 Controlo remoto de navegação (mudar dashboards).
- 🔐 Bloqueio remoto do Kiosk e gestão de ecrã.
- ⏲️ Atualização de Dashlets nativos no protetor de ecrã.

## 🚀 Instalação via HACS
1. Sobe estes ficheiros para um repositório no teu **GitHub**.
2. No Home Assistant, vai a **HACS > Integrações**.
3. Clica nos três pontos (canto superior direito) > **Custom repositories**.
4. Cola o URL do teu repositório e escolhe a categoria **Integration**.
5. Clica em **Instalar**.
6. Reinicia o Home Assistant.

## ⚙️ Configuração
1. Vai a **Definições > Dispositivos e Serviços**.
2. Clica em **Adicionar Integração** e procura por **HAKiosk**.
3. Insere o **IP do Tablet** (vê nas definições da app Android) e dá um nome ao dispositivo.
